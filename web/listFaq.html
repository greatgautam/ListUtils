FAQ

  • Who can use ListUtils.com?
    Anyone with access to internet can use this website.
  • What is NoExcel solution?
    If you work with lists of numbers or text and do not want to use a Microsoft Excel or any other Spreadsheet software, you can use ListUtils.com. If you have access to the Internet, you do not need any software installation.

  • What is the technology behind ListUtils.com?
